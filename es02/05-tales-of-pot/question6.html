MUSIC ANALYSIS

Marijuana mixtape: which is the marijuana beat?

  • Description
  • Protocol
  • Data

Description

We aggregated in one final corpus all the songs retrieved in the precedent protocol. The visualization shows within all the songs considered which is the genre of music that are is more perceived as linked to marijuana either if it talks about it explicitly or not.

The height of each bar indicates the number of songs within that genre. The resulting bar chart illustrates that hip-hop has by far the greatest number of songs. The color value represents another information, that is how many sub-genres are contained in each genre. The more variated the bar color is, the more subgenres are present. We can see that all the songs within Blues are not mixed with other subgenres.

Protocol

In Excel we created a corpus obtained combining the three lists from our three sources, cleaning the doubles. We manually searched again all the songs in Genius and Wikipedia to assign them a music genre. Then we aggregated again the genres in order to obtain 8 macro genres.
