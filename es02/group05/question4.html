MUSIC ANALYSIS

How many songs result searching marijuana on Youtube?

  • Description
  • Protocol
  • Data

Description

What do you find if you search “marijuana” on Youtube? We assumed that the output of this research would have been several different songs with “marijuana” or its synonyms in the title and instead we found out that the majority of the results were not music at all. The visualisation shows the amount of videos relating to every category from the three queries we used to search, underlying the music category. The grid is created to gain a perspective of the total 300 videos.

Protocol

We used the three query “marijuana”, “cannabis” and “weed” retrieved from the first protocol in order to create a corpus of videos. We downloaded 100 videos for each query with the YTDT Video List, with iteration 2 - one iteration means 50 results - and ranked by relevance. In Excel we aggregated the three lists in one corpus and created a visualization counting how many videos of every Youtube category there were.

Data

Timestamp: 01/12/2016 - 05/12/2016

Data source: YTDT video list

Download data (241KB)

The gathered data are three sheets in an Excel file with 100 songs each, the datas include all the informations provided by Youtube even though only the categories has been used in this visualization. It has been made a pivot for each query in order to count how many videos for each category and then they have been aggregated and summed in one final pivot used to create the visualisation.
